## Jinu Workshop

Welcome to Jinu Workshop, my personal portfolio website where I showcase my passion for Gunpla (Gundam plastic model kits), photography, and creative projects.

## About the Project

Jinu Workshop is a React-based website built to present:

- My Gunpla builds (standard & custom-painted)
- My photography work (indoor, outdoor, creative scenes)
- Insights into my hobby journey and collection
- A contact page to reach me easily

This project focuses on both design and user experience, with smooth animations and a clean layout.

- Tech Stack
- React (Vite)
- CSS (custom styling)
- React Router (navigation)
- Modular component structure

## Goals

Build a clean and modern portfolio
Practice React component architecture
Improve UI/UX design skills
Share my passion with a wider audience
